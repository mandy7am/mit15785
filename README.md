# 🧭 Sloan Compass: MIT Course Planner

**Sloan Compass** is an AI-curated scheduling platform designed for students navigating the complexities of MIT Sloan and Harvard cross-registration. This tool simplifies the journey from degree requirements to a balanced, personalized semester.

## 🚀 Key Features
* **Curated Bundles:** One-click course groupings for specialized tracks like Sustainability, Tech Leadership, and Finance.
* **Degree Audit Integration:** Real-time tracking of credit requirements.
* **Intelligent Scheduling:** AI-assisted suggestions to resolve cross-campus timing conflicts.
* **Institutional Branding:** Tailored UI reflecting the MIT and Harvard academic environment.

## 🛠️ Tech Stack
* **Frontend:** React, TypeScript, Tailwind CSS
* **Framework:** Vite
* **Design & Prototyping:** Lovable (Vibe Coding)
* **Refinement:** Cursor AI

## 🛠️ How to Run Locally
1.  Clone the repository: `git clone https://github.com/mandy7am/mit15785.git`
2.  Install dependencies: `npm install`
3.  Start the development server: `npm run dev`
